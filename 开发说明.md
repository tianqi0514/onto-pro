# Onto Pro 开发说明

## 当前版本

当前是一期开发骨架：

- 前端：React + TypeScript + Vite；
- 后端：Python + FastAPI + Pydantic；
- 数据：本地 JSON/mock；
- OCR：人工标注结构模拟；
- 数据库、对象存储、外部查询接口：已预留，功能标记开发中；
- 模拟推演：已预留入口、接口和 mock 模板。

## 本地启动

安装依赖：

```bash
python3 -m venv .venv
.venv/bin/pip install -r backend/requirements.txt
npm install
```

启动后端：

```bash
.venv/bin/python -m uvicorn backend.app.main:app --reload --host 0.0.0.0 --port 8001
```

启动前端：

```bash
npm run dev
```

访问：

```text
http://localhost:5173
```

## 验证

```bash
.venv/bin/python -m pytest backend/tests
npm run build
```
